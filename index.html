Coursework

CSC263

Data Structures and Analysis
Grade: A

Algorithm Analysis. Expected worst-case complexity, randomized quicksort. Standard abstract data types, such as graphs, dictionaries, priority queues, and disjoint sets. A variety of data structures for implementing these abstract data types, such as balanced search trees, hashing, heaps, and disjoint forests.

CSC309

Programming on the Web
Grade: A

An introduction to software development on the web. Operational concepts of the internet and the web, static client content, dynamic client content, dynamically served content, n-tiered architectures, web development processes, and security on the web.

CSC258

Computer Organization
Grade: A-

Computer structures, machine languages, instruction execution, addressing techniques, and digital representation of data. Computer system organization, memory storage devices, and microprogramming. Block diagram circuit realizations of memory, control and arithmetic functions.

CSC148

Introduction to Computer Science
Grade: A+

Abstract data types and data structures for implementing them. Linked data structures. Encapsulation and information-hiding. Object-oriented programming. Specifications. Analyzing the efficiency of programs. Recursion.

CSC108

Introduction to Computer Programming
Grade: A+

Programming in a language such as Python. Elementary data types, lists, maps. Program structure: control flow, functions, classes, objects, methods. Algorithms and problem solving. Searching, sorting, and complexity. Unit testing.

CSC209

Software Tools and Systems Programming
Grade: B+

Software techniques in a Unix-style environment, using scripting languages and a machine-oriented programming language (typically C). What goes on in the operating system when programs are executed. Core topics: creating and using software tools, pipes and filters, file processing, shell programming, processes, system calls, signals, basic network programming.

CSC165

Mathematical Expression and Reasoning for Computer Science
Grade: A

Informal introduction to logical notation and reasoning. Structuring proofs. Problem-solving techniques. Running time analysis of iterative programs. Formal definition of Big-Oh. Diagonalization, the Halting Problem. Unified approaches to programming and theoretical problems.

CSC207

Software Design
Grade: A-

An introduction to software design and development concepts, methods, and tools using a statically-typed object-oriented programming language such as Java. Topics from: version control, unit testing, refactoring, object-oriented design and development, design patterns, regular expressions.

CSC236

Introduction to the Theory of Computation
Grade: B+

The application of logic and proof techniques to Computer Science. Mathematical induction; correctness proofs for iterative and recursive algorithms; recurrence equations and their solutions; introduction to automata and formal languages.
